How to Generate Real Content

Use Mavis CLI to generate content (requires Matrix MCP configured):

# Image Generation
mavis mcp call matrix matrix_generate_image '{"prompt": "your description here"}'

# Video Generation
mavis mcp call matrix matrix_gen_videos '{"prompt": "your description here"}'

# Audio Synthesis
mavis mcp call matrix matrix_synthesize_speech '{"text": "your text here", "voice": "zh-CN"}'

# Music Generation
mavis mcp call matrix matrix_batch_text_to_music '{"requests": [{"prompt": "background music style", "duration": 10}]}'
